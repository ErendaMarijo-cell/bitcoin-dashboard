About Dashboard Traffic Data

The traffic metrics displayed here are based exclusively on anonymized and aggregated access data (e.g. number of requests per defined time interval).

No personal data is stored, no user profiles are created, and no external tracking or analytics services are used.
All data collection is performed exclusively on the server side and is intended solely for statistical analysis of overall dashboard usage.

Individual users are not identifiable at any time. The values shown do not represent individual user activity,
but only aggregated access frequencies within predefined time windows.
